Line Following Robot

Using Infrared Sensors, geared DC motors, raspberry pi, and additional components, created a line following robot.
The robot consisted 6 sensors in the front, each having its unique role.

Key feature added: The robot analyses the path it's being following for a while, and accordingly manages its speed. If it has been seeing a straight line for a while, it accelerates and upon noticing a curve slows down again. This is done with the help of the 2 extra sensors (The instructor asked to use only 4 sensors, we took permission to add 2 more, to add this feature.) which monitor anu change in path coming up ahead, and hence are placed a few centimeters ahead of the 4 sensors monitoring the path. It could also be controlled remotely through a mobile app.

Electronic Voting Machine

Using a fingerprint scanner, camera, RFID card reader, Raspberrypi, touchscreen UI and additional components (electronic and 3d printed) created a fully functional portable Electronic Voting Machine.

Working: An Admin can register a user through an online interface, create candidates, and start the voting process. Once the process has started, the voters go to the EVM, enter their username and password, scan their fingerprint and then go through a facial recognition test to cast a vote. The votes are then available for the admin to monitor real time.

Tech used: - Node js is the run-time environment for the backend.
- Used Express framework to serve backend requests for its simple and flexible structure, appropriate for rapid prototyping.
- Used the Flask framework to make the a facial recognition service. We do this since python has more libraries for accurate facial recognition.
- We use passport.js as authentication middleware for security purpose (logging in and signing up)
- We further take several security precautions such as storing hashed passwords instead of actual passwords as well as using cookies to maintain sessions for the online portal
- We used MongoDB as our database for the users’ information and the polling data
- Implemented with HTML, Javascript, CSS and handlebars, it also uses some style sheets from an online style sheet provider W3SCHOOLS.com
- We also used handlebars for rapid templating for HTML and CSS

Done as a Lab project over a period of 1 month at University of British Columbia, Vancouver.
Course Name: CPEN 291 : Computer Engineering Design Studio
Date: March 2019 - April 2019
Partners: Soroush Amirizadeh, Yash Dhandhania, Luqman Hakim, John Marquez and Thong Nguyen
My role/focus area: Developed the front-end website for the UI. Set up all the hardware along with Soroush.
